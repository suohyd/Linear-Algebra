Remark 7.2.10.

Suppose that \(v\in V\) is an eigenvector of \(T\) corresponding to eigenvalue \(\lambda\in F\text{.}\) If we let \(S=T-\lambda\unit_V\) then \(S(v)=0\text{,}\) i.e., \(v\) is an eigenvector corresponding to eigenvalue \(0\) for \(S\text{.}\)
Observe that if the matrix representation of \(T\) with respect to a basis \(\mathfrak{B}\) is \(A\) then, the matrix representation of \(T-\lambda\unit_V\) with respect to \(\mathfrak{B}\) is \(A-\lambda I_n\text{.}\) By CorollaryÂ 7.2.9, \(\det(A-\lambda I_n)=0\text{.}\) Hence, \(\lambda\) is an eigenvalue of \(T\) if and only if \(\det(A-\lambda I_n)=0\text{.}\)
