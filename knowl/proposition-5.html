Proposition 5.6.7.

Let \(V\) be a finite-dimensional vector space over a field \(F\text{.}\) We have the following.
  1. A vector \(v\in V\) is zero if and only if \(f(v)=0\) for every \(f\in V^*\text{.}\)
  2. If \(v\in V\) is such that \(f(v)\neq 0\) for some linear function \(f\in V^*\) then,
    \begin{equation*} V=\langle v\rangle\bigoplus\ker(f). \end{equation*}
  3. Consider nonzero linear functions \(f,g\in V^*\text{.}\) Then, \(\ker(f)=\ker(g)\) if and only if there exists a scalar \(\alpha\) such that \(f=\alpha g\text{.}\)
